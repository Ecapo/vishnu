Additional constraints

The following constraints apply to the element attributes, additional to the element attributes marked as required. A violation of any of these constraints will result in an unsuccessful result with the error code _InvalidResponse.
  • At least one of these attributes must be set: fromMachineId, toMachineId.
  • Either all of these attributes must be set, or none of them can be set: fromMachineId, fromUid.
  • Either all of these attributes must be set, or none of them can be set: toMachineId, toUid.
  • At least one of these attributes must be set: status with the value 'failed', failure.
