Function writeFile

Create a file or a directory. Content is required if it's a file. Existing files are always rewrited. Existing directories throws DirectoryAlreadyExists.

Input section

Input parameters

ParameterTypeDescriptionRequired
machineIdMachineIdMachine hash keyyes
path_textFile path.yes
isDirectory_booleanTrue if it's a directoryyes
content_base64File content.no
uidUidUsernameyes
passwordPasswordPasswordyes

Additional constraints

The following constraint applies to the input parameters, additional to the input parameters marked as required. A violation of this constraint will result in an unsuccessful result with the error code _InvalidRequest.
  • At least one of these parameters must be set: isDirectory with the value 'false' or content.
